class platform_input_support.helpers.download.AbortableStreamWrapper(stream, *, abort: Event)[source]¶

A wrapper around a stream that can be aborted.

This class wraps a stream and in every read operation, it will raise a platform_input_support.util.errors.TaskAbortedError if the abort event is set. This is useful to abort downloads when another task fails.

class platform_input_support.helpers.google.GoogleHelper[source]¶

Google Cloud Storage helper class.

This class provides methods to interact with Google Cloud Storage. It stores the Google Cloud Storage client and credentials.

  • client (google.cloud.storage.client.Client) – The Google Cloud Storage client.
